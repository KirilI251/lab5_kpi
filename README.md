# Лабораторна робота №5: Автоматизація тестування з Selenium WebDriver

**Виконав:** студент ІІІ курсу групи КП-32, Підгірний Кирило
**Перевірив:** Олексій Артур

Цей репозиторій містить автоматизовані тести для веб-додатку [the-internet.herokuapp.com](https://the-internet.herokuapp.com/), виконані в рамках лабораторної роботи Варіант 4.

## Опис
Проєкт реалізовано мовою **C#** з використанням фреймворку **NUnit** та бібліотеки **Selenium WebDriver**. 
Архітектура проєкту базується на використанні базового класу `SeleniumTests` для керування життєвим циклом браузера (Setup/Teardown) та налаштування ChromeOptions.

## Технології
* **Мова:** C# (.NET 9.0)
* **Тестовий фреймворк:** NUnit
* **Інструмент автоматизації:** Selenium WebDriver
* **Браузер:** Google Chrome
* **Додаткові бібліотеки:** DotNetSeleniumExtras.WaitHelpers (для явних очікувань)

## Реалізовані тести (Варіант 4)

В проєкті автоматизовано 10 сценаріїв тестування згідно з завданням:

1. **Add/Remove Elements** (`/add_remove_elements/`)
    * Додавання нового елемента (кнопки "Delete").
    * Перевірка появи та подальшого видалення елемента.
2. **Checkboxes** (`/checkboxes`)
    * Перевірка початкового стану чекбоксів (Checked/Unchecked).
    * Перевірка зміни стану чекбокса після кліку.
3. **Dropdown** (`/dropdown`)
    * Вибір опцій з випадаючого списку за текстом ("Option 1", "Option 2").
    * Перевірка, що опція успішно обрана.
4. **Inputs** (`/inputs`)
    * Введення числових значень у поле вводу.
    * Перевірка правильності збереженого значення атрибуту `value`.
5. **Status Codes** (`/status_codes`)
    * Перехід на сторінку статусу 200.
    * Перевірка наявності тексту про успішний статус код (200 OK).
6. **Drag and Drop** (`/drag_and_drop`)
    * Емуляція перетягування елемента А на місце Б.
    * Використання JavaScript workaround для обходу обмежень HTML5 Drag&Drop.
7. **Shifting Content** (`/shifting_content`)
    * Перехід до розділу "Menu Element".
    * Перевірка кількості елементів меню (5 пунктів) попри зсув контенту.
8. **Geolocation** (`/geolocation`)
    * Перевірка роботи кнопки "Where am I".
    * Використання CDP (Chrome DevTools Protocol) для підміни (Mocking) координат на фіктивні (Лондон) задля приватності.
9. **JavaScript Errors** (`/javascript_error`)
    * Завантаження сторінки з навмисною помилкою.
    * Перевірка логів браузера на наявність повідомлення про помилку JS.
10. **Exit Intent** (`/exit_intent`)
    * Емуляція руху миші та події виходу курсору за межі вьюпорта (через JS).
    * Перевірка появи модального вікна при спробі закрити вкладку.
